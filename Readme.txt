To run the project files in Unity:

1.Download the files in a folder for example :- Killer Box.
2.Then open Unity project window and then click on open button.
3.Then it will show a pop up where you have choose the file, Choose the files where 
you have downloaded the project and then click open.
3.The project will be opened then click on Play to test it. 

To run on Android: 

1.To test the game directly transform the Killer Box.apk file to you android phone
2.Then Install it during the process of installation  it will ask permission to install  give the permission to install 
3.Once it gets Installed click open 
4.THEN ENJOY!
